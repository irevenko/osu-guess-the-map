# Quick Start 🚀
```git clone https://github.com/irevenko/osu-guess-the-map.git``` <br>
```cd osu-guess-the-map/client``` <br>
```npm run serve``` <br>
```cd osu-guess-the-map/server```  <br>
```node server.js```
```go to localhost:8080```
